# Project Statement : Inventory-Management-System

## 1. Problem Statement

The Inventory management system is a lightweight command-line python application that helps users manage product inventory using a simple CSV-based database.The program supports adding new items, updating stock and viewing all available products along with their prices. This project is perfect for beginners learning file handling, python functions and CSV manipulation.

## 2. Scope of the Project

The scope of the Inventory-Management-System is to deliver a fully functional *Minimum Viable Product (MVP)* focusing on the core functional requirements of managing the stocks of products in a store using *Python*.

*Inclusions :*
* User Data Input for Item Name, Item Brand, Stock Quantity and Cost
* Data Persistence : Storage and retrieval of Stocks and Product using simple CSV File Operations
* Basic Reporting : Displaying a summary of the stocks left and their respective prices
* Implementation using Python's Standard Library

*Exclusions:*
* Integration with external databases
* Advanced user authentication or multi-user support
* Complex visualisation or prediction features

## 3. Target Users

The primary target users for the Inventory Management System include *Small or Medium Sized Enterprises (SME)* which cannot spare a huge sum on simple stock management softwares

## 4. High-Level Features

* Core Logging (CRUD) : Fast, intuitive entry and retrieval of products in a store
* Data Persistence : Reliable Storage and Integrity of records using local CSV file storage

## 5. Future Goals

* Graphical User Interface : For ease in usage
* User authentication : Ensuring the stock updation is done by authorised personnals only
* Logging : Logging the sales and changes in the stocks
* Summary Statistics : Displaying the sales per product so as to find out the most sold product
